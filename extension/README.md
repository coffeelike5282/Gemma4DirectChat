# Gemma 4 Direct Connector

Native integration for Gemma 4 within the Antigravity IDE.

## Features
- Direct sidebar chat with local Ollama engine.
- Synthetic Ether design system (Glassmorphism).
- Advanced parameter controls (Temperature, Top K/P, etc.).

## Requirements
- Ollama running locally (`ollama serve`).
- Gemma 4 model installed (`ollama run gemma4:e2b`).

## How to use
1. Install the `.vsix` file.
2. Click the Gemma icon in the Activity Bar.
3. Start chatting with Gemma 4!
